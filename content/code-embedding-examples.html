Code Embedding Examples

Highlight.js makes it incredibly easy to embed and beautifully display code snippets for various programming languages within your documentation. Simply wrap your code in `<pre><code>` tags and specify the language using a class attribute.

Key Requirement

For Highlight.js to work, always ensure your code is wrapped within a `<pre><code>` block, and the `<code>` tag has a class in the format `class="language-xyz"`, where `xyz` is the language identifier (e.g., js, html, css, php, python).

1. JavaScript Example

Displaying JavaScript code for your functions or scripts.


// Function to add two numbers
function addNumbers(a, b) {
    return a + b;
}

// Example usage
const result = addNumbers(5, 3);
console.log(`The sum is: ${result}`);

// Async operation example
async function fetchData(url) {
    const response = await fetch(url);
    const data = await response.json();
    return data;
}

2. HTML Example

Embedding HTML markup, remember to escape `<` as `&lt;` and `>` as `&gt;`.


<!-- A simple navigation bar -->
<nav class="navbar navbar-expand-lg navbar-light bg-light">
    <div class="container-fluid">
        <a class="navbar-brand" href="#">MySite</a>
        <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav" aria-controls="navbarNav" aria-expanded="false" aria-label="Toggle navigation">
            <span class="navbar-toggler-icon"></span>
        </button>
        <div class="collapse navbar-collapse" id="navbarNav">
            <ul class="navbar-nav ms-auto">
                <li class="nav-item">
                    <a class="nav-link active" aria-current="page" href="#">Home</a>
                </li>
                <li class="nav-item">
                    <a class="nav-link" href="#">Features</a>
                </li>
            </ul>
        </div>
    </div>
</nav>

3. CSS Example

Showcasing custom CSS rules or Bootstrap overrides.


/* Custom button styling */
.btn-custom {
    background-color: #6c5ce7;
    color: #ffffff;
    border-radius: 5px;
    padding: 10px 20px;

.btn-custom:hover {
    background-color: #483d8b;
    transform: translateY(-2px);

/* Responsive adjustments */
@media (max-width: 768px) {
    .container {
        padding: 0 15px;
    }
}

4. PHP Example

If your project involves backend logic, here's how to display PHP code.


<?php

// Define a function to greet a user
function greetUser($name) {
    return "Hello, " . htmlspecialchars($name) . "!";
}

// Get name from query parameter, or default
$userName = $_GET['name'] ?? 'Guest';

// Output the greeting
echo greetUser($userName);

// Simple loop
for ($i = 0; $i < 5; $i++) {
    echo "<br>Number: " . $i;
}

?>

5. Python Example

For Python-based scripts or snippets.


# Function to calculate factorial
def factorial(n):
    if n == 0:
        return 1
    else:
        return n * factorial(n-1)

# Example usage
num = 5
print(f"The factorial of {num} is: {factorial(num)}")

# Class definition
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def bark(self):
        print(f"{self.name} says Woof!")

my_dog = Dog("Buddy", 3)
my_dog.bark()

6. Plain Text / No Specific Language

You can also use language-plaintext for generic text or configuration files where no specific syntax highlighting is needed, but you want to maintain formatting.


# This is a configuration file example
# Lines starting with # are comments

[database]
host = localhost
port = 5432
user = admin
password = your_password

[application]
